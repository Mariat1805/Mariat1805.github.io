Healthcare Data Analysis

MySQL • Python • Data Cleaning • EDA

End-to-end SQL analysis of a synthetic healthcare dataset covering 55,000 patient records, including data cleaning in Python and exploratory analysis in MySQL.

  • Problem: Analyze patient demographics, billing patterns, and clinical outcomes across a large healthcare dataset

Project 04

Brazilian E-Commerce Analysis

MySQL • Python • Multi-table JOINs • CTEs • Window Functions

End-to-end analysis of a real Brazilian e-commerce dataset (Olist) with 8 relational tables and 500,000+ rows, covering revenue trends, delivery performance, and customer geography.

  • Problem: Identify revenue drivers, delivery bottlenecks, and customer distribution across Brazil
  • Approach: Cleaned 8 tables in Python, modelled relationships in MySQL, and wrote multi-table JOINs, CTEs, and window functions to answer business questions
  • Insight: Beauty & Health topped revenue despite not having the most orders; remote northern states averaged 29+ days delivery vs ~10 days for São Paulo; November 2017 showed a clear Black Friday spike
